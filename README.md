# LogiSafe-DevSecOps

Backend: Python Flask
Containerization: Docker
Orchestration: Kubernetes (Minikube)
CI/CD: Jenkins
Monitoring: Prometheus & Grafana
Security: SonarQube, Trivy
IaC: Terraform

LogiSafe is a centralized log management and DevSecOps monitoring platform that collects logs from containerized applications deployed on Kubernetes. It integrates CI/CD automation, security scanning, and real-time monitoring to ensure reliable and secure software delivery.
